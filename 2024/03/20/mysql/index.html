MYSQL学习笔记

JoBackRer2024-03-202024-03-28

一.SQL语句

n. -> 数据库/表名称
v. -> 附加属性

附加属性 解释
null 数据可空
notnull 数据不可空
primary key 设置主键
auto_increment 自增
unique 不能出现重复

1.数据库管理

create database n.; 创建数据库
drop database n.; 删除数据库
show databases; 查看所有数据库
use n.; 选择数据库

2.数据表管理

创建数据表
create table n.(
//列信息
字段名称 数据类型() v.,
字段名称 数据类型() v.
);
drop table m.;删除表
show tables; 查询当前所有数据表
desc n.; 表类型

修改数据表结构，修改字段

命令 解释
alter table n. add 字段名称 数据类型 新增字段
alter table n. change 旧字段名称 新字段名称 数据类型 修改字段名称
alter table n. drop 字段名称 删除字段
alter table n. alter column 字段名称 set v. 值 修改字段附加属性
alter table n. modify 字段名称 数据类型 附加属性 修改字段的类型以及附加属性

3.数据管理

insert into n.(字段名称1,字段名称2,…) values(值1,值2,…),(值3,值4,…); 新增数据
insert into n. values(所有字段值); 新增字段

delete from n.; 删除表内所有数据
delete from n. where 字段名称 = 值; 删除指定字段值的整行字段

update n. set 字段名称 = 值,字段名称 = 值,…;直接修改所有提供字段的值
update n. set 字段名称 = 值,字段名称 = 值,… where 字段名称 = 值; 修改指定字段的值

select 字段,字段,… from n.; 查询来自n.的字段所有值*表示所有字段
select 字段列表 from n. where 字段名称 = 值; 指定查询

4.主键

表中的一个字段，该字段的值是每一行数据的唯一标识

默认情况，每张表都要有一个主键，也只能有一个主键

主键生成错略：代理主键，与业务无关的字段，仅仅是用来标识一行数据，一般定义为int类型

主键必须是唯一的

二.数据类型

1.整数类型

2.小数类型

3.日期和时间类型

4.字符串类型

5.二进制类型

SQL函数

数学函数，日期函数，字符串函数，聚合函数
函数方法加在字段名称与表上
如下案例：

数学函数

绝对值abs()
向下取整floor()
向上取整ceil()

字符串函数

insert(s1,index,len,s2)
s1是目标字符串，index是下标(从1开始)，len上添加长度，s2是追加内容
upper()字母转大写
lower()字母转小写
left(s,len)返回s字符串前的len个字符
right(s,len)返回s字符串后的len个字符
substring(s,index,len)截取s字符串，从index开始，长度为len字符
reverse()反方向阅读

日期函数

curdate()获取日期函数
curtime()获取时间
now()获取当前时间
datediff(d1,d2)日期加n天后的日期
subdate(d,n)和上个方法相反

聚合函数

count()根据某个字段统计总记录数
sum()计算某个字段值总和
avg()求某个字段值的值平均值
max()求某个字段值最大值
min()求某个字段值最小值

特殊查询

分组查询

select 字段组 from n. group y 字段名称; 分组查询字段名称的字段值

排序查询

select * from n. order by 字段名称 desc; 降序查询字段值排序
select * from n. order by 字段名称 asc; 升序查询字段值排序

运算符

执行运算符：加减乘除
比较运算符：大小等于 1为true 0为false
逻辑运算符：与或非
特殊运算符：is null判断空 between n and n在两者之间 in(区间)是否在这区间

like模糊查询
select 字段组 from n. where name like ‘%值%’ 模糊查询值
%表示代表任意值
